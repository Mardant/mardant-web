Privacidad

Política de privacidad

En Mardant respetamos la privacidad de nuestros clientes y utilizamos sus datos únicamente para fines relacionados con la atención comercial, gestión de pedidos, pagos, comprobantes y entregas.

Datos que podemos solicitar

  • Nombre o datos de contacto necesarios para coordinar una compra.
  • Número de WhatsApp o teléfono de contacto.
  • Datos necesarios para emitir boleta de venta.
  • Dirección, agencia, distrito o punto de entrega para coordinar envíos.
  • Información relacionada con el pedido, reserva, preventa o producto solicitado.
  • Comprobantes de pago enviados voluntariamente por el cliente.

Uso de la información

  • Atender consultas por WhatsApp.
  • Confirmar compras, reservas, preventas y productos a pedido.
  • Coordinar pagos y validación de abonos.
  • Emitir boletas de venta.
  • Gestionar envíos mediante operadores logísticos.
  • Atender reclamos, quejas o solicitudes relacionadas con una compra.
  • Mejorar la experiencia de navegación en la web.

Cuenta de cliente

La sección “Mi Cuenta” utiliza un identificador interno de cliente y contraseña. Esta sección no solicita directamente datos como DNI, dirección o teléfono dentro del formulario de acceso.

Envíos y terceros

Para realizar entregas, Mardant puede compartir únicamente los datos necesarios con operadores logísticos como Shalom, Olva Courier, servicios de delivery, motorizados u otros medios coordinados previamente con el cliente.

Google Analytics y cookies

La web puede utilizar herramientas como Google Analytics para obtener información general y estadística sobre el uso del sitio. Esta información ayuda a mejorar la navegación, el rendimiento y la experiencia del usuario.

Derechos del cliente

El cliente puede solicitar información, actualización, corrección o eliminación de sus datos escribiendo a: gamesmardant@gmail.com.
